# Asteroids (minimalistic)
## Description:
This is a basic program which will run a minimalistic game of Asteroids.\
This program was written as part of a guided project on [Boot.dev](https://www.boot.dev/dashboard) for learning python and pygame

## Features
- Hit detection
- Asteroid splitting

## Technologies Used
- Python 3.12
- Pygame 2.6.1
- Visual Studio Code

## Installation and Setup
- Create and activate a new virtual environment in a directory of choice
- Add pygame library as a project dependency
- Download all project files to virtual environment

## Usage
- run program with command: uv run main.py
- use wasd keys to move and spacebar to shoot
